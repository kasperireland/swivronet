Download YouTube videos for free (SaveMyVids Release)

By Dmitri
July 17, 2021

Hi there, SaveMyVids is a new project just released by Swivro. It is based off of YouTube Downloader, and uses the YouTube-DL Library for downloading YouTube videos and ElectronJS to run as a desktop application.  It's 100% free, and is made available publicly, under the Public Domain, with the source code being on GitHub.

If you don't care about the code or don't want to look at it, you can download the latest release directly, on the SaveMyVids website, at http://savemyvids.uk.to where you can also find screenshots of the application running. Stop paying for that YouTube Premium garbage to download YouTube videos, and just use SaveMyVids. As mentioned, it’s free and runs on desktop platforms.

This post is short because SaveMyVids is a small project we decided to create just for fun, there isn't much to it, and it won't be updated constantly. Thanks for reading.
